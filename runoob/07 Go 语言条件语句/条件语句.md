
Go 编程语言中 switch 语句的语法如下：
```text
switch var1 {
    case val1:
        ...
    case val2:
        ...
    default:
        ...
}
```

switch 语句还可以被用于 type-switch 来判断某个 interface 变量中实际存储的变量类型。
Type Switch 语法格式如下：
```text
switch x.(type){
    case type:
       statement(s);      
    case type:
       statement(s); 
    /* 你可以定义任意个数的case */
    default: /* 可选 */
       statement(s);
}
```
